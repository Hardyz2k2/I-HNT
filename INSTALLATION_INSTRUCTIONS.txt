ğŸ› ï¸ TROUBLESHOOTING
==================

Problem: "python is not recognized"
Solution: Python is not in PATH
- Reinstall Python and check "Add Python to PATH"
- Or manually add Python to PATH in System Environment Variables

Problem: "pip is not recognized"
Solution: Install pip manually
- Download get-pip.py from https://bootstrap.pypa.io/get-pip.py
- Run: python get-pip.py

Problem: "Permission denied" during pip install
Solution: Run Command Prompt as Administrator
- Right-click Command Prompt â†’ "Run as administrator"
- Try pip install again

Problem: "No module named 'cv2'" or similar
Solution: Package installation failed
- Try: pip install --upgrade pip
- Then: pip install -r requirements.txt
- If still failing, install packages individually

Problem: "CUDA not available" (for GPU acceleration)
Solution: This is normal if you don't have NVIDIA GPU
- The application will work with CPU (slower but functional)
- For GPU acceleration, install NVIDIA drivers and CUDA toolkit
